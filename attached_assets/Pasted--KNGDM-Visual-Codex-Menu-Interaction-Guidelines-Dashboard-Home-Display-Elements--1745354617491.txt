# KNGDM Visual Codex - Menu Interaction Guidelines

## Dashboard / Home

### Display Elements
- **Recent Projects Grid**
  - Card layout with thumbnail preview
  - Brand name and last modified date
  - Completion percentage indicator
  - Quick access button to continue editing

- **Analytics Summary**
  - Usage statistics card
  - Weekly activity graph
  - Popular sections accessed
  - Brand consistency score

- **Template Gallery**
  - Horizontal carousel of available templates
  - Category filtering tabs
  - Preview thumbnails with hover zoom
  - "Use Template" and "Preview" buttons

### User Interactions
- **Hover Effects**
  - Project cards: Subtle elevation (2px rise)
  - Action buttons: Background color shift to `sidebar-hover`
  - Templates: Scale to 105% with shadow enhancement

- **Click Actions**
  - Project card: Opens project in last edited section
  - Create new: Opens template selection or blank project
  - Analytics card: Expands to detailed analytics view
  - Template: Shows full-screen preview with "Use" option

- **User Operations**
  - Create new brand guide (+ button)
  - Delete project (via context menu)
  - Duplicate project (via context menu)
  - Pin favorite projects to top
  - Filter projects by status/date
  - Export analytics reports

## Global Interaction Patterns

### Common Save Operations
- **Auto-save**
  - Real-time saving (indicated by subtle checkmark)
  - 5-second delay after changes stop
  - Version history point created every 15 minutes
  - Offline changes cached and synced when reconnected

- **Manual Save**
  - Save button in action bar (persistent)
  - Keyboard shortcut: Ctrl/Cmd + S
  - Named save points with descriptions
  - Save confirmation with version note option

### Universal Actions
- **Global Keyboard Shortcuts**
  - Esc: Close dialogs, cancel operations
  - Ctrl/Cmd + Z: Undo last action
  - Ctrl/Cmd + Shift + Z: Redo action
  - Ctrl/Cmd + F: Focus search
  - Ctrl/Cmd + /: Open keyboard shortcut help

- **Context Menus**
  - Right-click on any element shows relevant actions
  - Section headers include section-specific actions
  - Asset right-click offers download/edit options
  - Text elements offer copy/edit/generate options

### State Management
- **Loading States**
  - Skeleton screens instead of spinners
  - Progressive loading of content
  - Background operations indicator in header
  - Cancel option for long-running operations

- **Error Handling**
  - Inline error messages with recovery options
  - Auto-retry for connection issues
  - Error notifications with detailed information
  - Guided recovery steps for complex errors

### Feedback & Notifications
- **System Notifications**
  - Toast messages for completed operations
  - Banner alerts for important system messages
  - In-app notification center with history
  - Email notifications for critical events

- **User Feedback Collection**
  - Reaction buttons on generated content
  - Improvement suggestions for AI outputs
  - Feature request button in footer
  - Satisfaction pulse surveys
